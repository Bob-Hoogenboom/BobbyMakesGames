Cuphead

Forest Follies stage remake demo


This is a small demo me and 6 other people worked on. Its a small recreation of the stage 'Forest Follies' from 'Cuphead'

Lead-Developer

My role in this project was the lead-developer
I mostly did the enemies and i implemented the art from our 4 artists, i also managed the github page and helped where i could.

Player-Animations

Part of making the enemies was adding animations to the enemies via parameters.

Here you can see the animation tree for the piranha plant above.

Tools

For this project we used Unity as our main engine, the sprites were made in Photoshop and all added into sprite sheets for easy animating in Unity. we also used Visual Studio Code as our main code editor for this project.
