Zelda Ocarina of Time

Zelda puzzle room project


This is a small project i did with 2 other team members. We picked out a game we all liked and tried to recreate a part of that game. We choose 'Ocarina of Time 64'and we recreated the water puzzle from the first dungeon, 'Inside the great Deku tree'. We didnt really have a lead Developer because we all knew where we wanted to push the game Demo.

Player-Movement

My main task for this project was to make a character controller.
I used a Unity Rigidbody character controller because we wanted to apply some sort of force to link in the form of knockback.

Here you can see how i made gravity for our player character so the player can jump.

Player-Animations

Another task of me for this project was to give the player animations.
I did this using mixamo animations for humanoids and Unity's animation system.

Here you can see the animator

Tools

We used Unity as a main engine for this project, Blender to model the room and assets, Mixamo for animations and Visual Studio Code for editting our code in Unity.
